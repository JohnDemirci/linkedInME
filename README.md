# LinkedInME

A sample iOS application demonstrating the Supervision state management architecture.

---

## Requirements

- **Xcode** — Latest version installed
- **Apple Intelligence** — Must be enabled on your device

---

## Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Open the project

Open the project in Xcode.

### 3. Trust the Swift macros

When you first build, you may see a prompt about macros from the Observation framework. Click **Allow** or **Trust** to let the system accept the macro-generated code from `@Observable`.

> **Note:** This is expected behavior. Supervision uses Swift macros for state observation and this prompt only appears on the first build.

### 4. Build and run

Press `⌘ R` to build and launch the application powered by Supervision.
